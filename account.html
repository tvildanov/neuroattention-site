Напоминание в вашем календаре

Шаг 1 из 5

NeuroAttention — программа строится вокруг вашей персональной NeuroMap — карты внимания, которая отражает текущее состояние и динамику изменений. Вы проходите адаптационный тест, получаете профиль и начинаете работу с практиками, адаптированными под ваше состояние.

Каждая практика — это сеанс с аудиопрактикой, направленный на развитие конкретного навыка внимания. По окончании записываются ваши ощущения, создаётся динамический граф.

На протяжении программы ваша карта растёт и раскрывает скрытые связи: как события влияют на внимание, как эмоции запускают мысли, как всё это приводит к действиям.

📋 Как подключить Cloudflare R2 (рекомендуется для больших объёмов)
  1. Зарегистрируй Cloudflare аккаунт (бесплатно, R2 идёт без egress-сборов)
  2. R2 → Create bucket (например neuroattention-media)
  3. Manage R2 API Tokens → создай Read+Write токен. Сохрани Access Key ID + Secret
  4. Settings бакета → Public access → Allow Access. Запиши Public Bucket URL (либо подключи свой домен media.neuroattention.org)
  5. В Railway-переменных neuroattention-api добавь:
    • R2_ACCOUNT_ID
    • R2_BUCKET
    • R2_ACCESS_KEY_ID
    • R2_SECRET_ACCESS_KEY
    • R2_PUBLIC_BASE_URL — например https://pub-xxx.r2.dev или https://media.neuroattention.org
  6. Railway передеплоит. Все новые загрузки автоматически пойдут в R2. Старые файлы на GitHub Pages продолжат работать без миграции.

Стоимость R2: $0.015/GB-месяц хранения, $0 egress. Первые 10 GB бесплатно. Файл 1 GB MP4 в месяц = $0.015. Для сравнения: GitHub Pages начинает предупреждать на 5 GB репозитория.
